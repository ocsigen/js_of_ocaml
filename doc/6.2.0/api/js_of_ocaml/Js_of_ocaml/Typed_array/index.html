  2. | Int8_unsigned : (int, int, Stdlib.Bigarray.int8_unsigned_elt) kind
  3. | Int16_signed : (int, int, Stdlib.Bigarray.int16_signed_elt) kind
  4. | Int16_unsigned : (int, int, Stdlib.Bigarray.int16_unsigned_elt) kind
  5. | Int32_signed : (Js.number_t, Stdlib.Int32.t, Stdlib.Bigarray.int32_elt) kind
  6. | Int32_unsigned : (Js.number_t, Stdlib.Int32.t, Stdlib.Bigarray.int32_elt) kind
  7. | Float32 : (Js.number_t, float, Stdlib.Bigarray.float32_elt) kind
  8. | Float64 : (Js.number_t, float, Stdlib.Bigarray.float64_elt) kind

The first type parameter is the type of values that can be read and written in the typedArray. The last two type parameters define the kind of bigarrays that can be converted to and from the typedArray. See Stdlib.Bigarray.kind.
