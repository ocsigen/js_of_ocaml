The OCaml toplevel, running in a Web Worker

        

        

          The toplevel runs off the UI thread, so a long computation never
          freezes the page. A Web Worker has no DOM though, so code typed here
          cannot touch the page (Dom_html, Graphics, ...); the stdlib and Lwt
          work.
